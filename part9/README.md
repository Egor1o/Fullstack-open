# PART 9
This is [part 9](https://fullstackopen.com/en/part9) of the Full Stack Open course.
Here is some brief information you might need before starting with each repository.

## Common
In each repository containing a `package.json`, first run `npm install` to install all the dependencies.
Then you can run the application with `npm start` or `npm run dev` for development mode.

### flight_diaries
For flight diaries to work, please use the forked backend repository with the required modifications.
The link is [here](https://github.com/Egor1o/flight-diary-backend). Clone the repository to the root 
directory of flight_diaries.

### patientor
If you would like to have a frontend for Patientor, please clone
[this](https://github.com/Egor1o/patientor) repository in the same root folder where
the backend is located.
